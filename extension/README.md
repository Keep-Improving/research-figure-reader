# 科研图片理解助手浏览器插件

这个插件会在论文网页或浏览器 PDF 阅读页中提供“解析图”入口，把选中的图、页面截图、caption 候选和上下文发送到配置的后端 API，再把红框和解释覆盖回原网页。

## 安装

1. 先启动后端，默认地址是 `http://127.0.0.1:8787`。
2. 在 Chrome/Edge 打开扩展管理页，开启开发者模式。
3. 点击“加载已解压的扩展程序”，选择本项目的 `extension` 目录。
4. 进入插件“详情”或“选项”，确认后端 API 地址。

## 配置后端地址

插件默认连接：

```text
http://127.0.0.1:8787
```

如果后端部署在服务器上，在插件 Options 页面填写公开后端地址，例如：

```text
https://your-api.example.com
```

Options 页面里的“测试连接”会请求 `/api/health`，用于确认后端可访问、模型名和 API key 配置状态。插件不会保存或暴露模型 API key。

## 公开给别人使用时

- 后端必须由服务端持有 `OPENAI_API_KEY`。
- 插件只配置后端地址，不直接调用模型服务。
- 如果要做公共网站/公共插件，需要加入用户登录、数据隔离、数据库存储和隐私说明。
