# 科研图片理解助手浏览器扩展 MVP

这个扩展把现有本地后端接到论文网页里。打开论文网页后，页面中的候选图片左上角会出现“解析图”按钮。点击后扩展会截取当前可见图片区域，调用本地 `http://127.0.0.1:8787/api/analyze-image`，并把返回的标注框覆盖回网页图片上，同时在右侧显示总解释。

## 使用方式

1. 先启动本项目后端：在 `app` 目录运行 `npm run server`。
2. 在 Chrome/Edge 打开扩展管理页，开启开发者模式。
3. 选择“加载已解压的扩展”，加载本目录 `extension`。
4. 打开论文网页，点击图片左上角的“解析图”。

## 当前限制

- 目前主要支持网页中的 `<img>` 图片。
- 浏览器内置 PDF 阅读器和 Zotero 还没有接入。
- 如果图片超出当前视口，扩展会分析当前可见部分；后续需要做整图滚动截图或直接读取原图资源。
- 标注框来自真实模型输出，不使用模拟结果；本地后端未启动或模型接口不可用时会直接报错。
